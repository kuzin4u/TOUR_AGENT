## 📌 Описание проекта

Проект представляет собой **полностью автономный HTML-дашборд**, который воспроизводит бизнес-модель турагентства «Формула Отдыха» на основе реальных данных из Excel. Все расчёты, формулы и взаимосвязи перенесены в интерактивный веб-интерфейс.

Инструмент позволяет:
- Редактировать объёмы, чеки и комиссии по продуктам (пакетные туры, FIT, MICE, авиабилеты, отели, трансферы, визы, страховки).
- Мгновенно пересчитывать выручку, переменные/постоянные затраты, ФОТ и EBIT.
- Включать/выключать влияние ИИ-мероприятий (рост конверсии, снижение времени на лид).
- Анализировать эффективность 9+ ИИ-мероприятий с фильтрацией по группам.
- Оценивать стоимость ручной работы без ИИ.
- Сравнивать конверсию MICE до и после внедрения ИИ (9% → 57%).
